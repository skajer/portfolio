# Instrukcje dotyczące obrazów projektu strony Nowa Droga

W tym folderze należy umieścić następujące obrazy:

1. strona-glowna.jpg - zrzut ekranu strony głównej witryny
2. sekcja-o-mnie.jpg - zrzut ekranu sekcji "O mnie" 
3. oferta.jpg - zrzut ekranu sekcji "Oferta"
4. wersja-mobilna.jpg - zrzut ekranu strony w wersji mobilnej
5. struktura-strony.jpg - diagram przedstawiający strukturę strony

Obrazy można uzyskać, wykonując zrzuty ekranu strony https://nowadroga-katarzyna-tokaj-czaja.pl/ lub pobierając je z oryginalnych zasobów projektu.

W razie braku oryginalnych obrazów, można tymczasowo zastąpić je zaślepkami (placeholder images) o wymiarach około 800x600 pikseli.

Zalecany format obrazów: JPG lub PNG z kompresją dla optymalnej szybkości ładowania strony.
