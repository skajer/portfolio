System automatyzacji domu

Kompleksowy system zarządzania inteligentnym domem oparty o Home Assistant

Home Assistant Node-RED MQTT

Przegląd projektu

Cel projektu

Stworzenie intuicyjnego i niezawodnego systemu automatyzacji domu, który pozwala na pełną kontrolę nad oświetleniem, ogrzewaniem, bezpieczeństwem i innymi aspektami życia domowego.

Technologie

  • Home Assistant
  • Node-RED
  • MQTT
  • ESP8266/ESP32
  • Docker

Główne funkcje

Inteligentne oświetlenie

Automatyczne sterowanie oświetleniem w zależności od pory dnia i obecności domowników.

Kontrola klimatu

Zarządzanie ogrzewaniem i klimatyzacją z uwzględnieniem preferencji użytkowników.

System bezpieczeństwa

Monitoring, czujniki ruchu i automatyczne powiadomienia o zdarzeniach.

Implementacja

Konfiguracja automatyzacji w Home Assistant


automation:
  - alias: "Włącz światło przy wejściu"
    trigger:
      platform: state
      entity_id: binary_sensor.motion_sensor
      to: 'on'
    condition:
      condition: sun
      after: sunset
    action:
      service: light.turn_on
      target:
        entity_id: light.entrance
      data:
        brightness_pct: 100
        transition: 2
